ERP Backend Setup

Configure your Google Apps Script web app URL

Setup Instructions:
  1. Open Google Apps Script (script.google.com)
  2. Create a new project and paste the backend.gs code
  3. Update the SHEET_ID constant with your Google Sheet ID
  4. Click Deploy → New Deployment → Web App
  5. Set "Execute as" = Me, "Who has access" = Anyone
  6. Copy the Web App URL and paste it below
This URL is stored in your browser's local storage and is only accessible on this device.
